// THE PROBLEM

지금도 위조 서류가
통과되고 있습니다

구직자 5명 중 1명이 허위 기재. 수기 검증으로는 절대 막을 수 없습니다.

// 01
감사 지적 = 담당자 개인 징계

위조 서류가 합격 후 발견되면 담당자 개인이 책임을 집니다. "눈으로 확인했다"는 말은 법정에서 통하지 않습니다.

832건/ 연간 공공기관 채용 비위 (감사원 2024)
// 02
2주 × 알바 15명 = 1,000만원

4,000건 공채 서류 검증에 인건비만 900~1,000만원. 피로 누적으로 인한 휴먼 에러는 막을 수가 없습니다.

1,000만원/ 4,000건 수기 검증 비용
// 03
업무 시간의 90%가 민원 전화

서류 미비 안내, 재제출 요청, 확인 전화. 채용 담당자가 실제 채용 업무를 할 시간이 없습니다.

90%/ 담당자 업무 중 민원 처리 비율
// 04
교묘해진 위조, 육안으로 불가

AI로 생성된 위조 서류는 전문가도 구분하기 어렵습니다. 발급기관 DB와 직접 대조하지 않으면 걸러낼 수 없습니다.

20%/ 구직자 허위 기재 비율
// HOW IT WORKS

입력 → 엔진 → 결과
단 3단계
