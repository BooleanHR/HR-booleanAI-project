### 1. 비즈니스 개요 및 레퍼런스

본 보고서는 채용 시장의 '신뢰 자본(Trust Capital)' 부족 문제를 해결하기 위한 **Legal-Tech 기반 AI 검증 솔루션**의 가치사슬을 설계합니다. 기존의 **단순 OCR 추출**이나 **수기 검증**이 가진 '증빙력 부재'와 '운영 병목' 문제를 **'원본-조회본 병렬 캡처'**와 **'Triple Check Loop'** 기술로 파괴하며, 인사 담당자에게 '실무 공수 제로'와 '법적 면책권'을 동시에 제공하는 운영 구조를 제안합니다.

### 4. 종합 분석 결과 및 실천 로드맵 (Insights)

### 🥇 핵심 가치창출 요인 (Value Drivers)

1. **담당자 면책권 확보 (Audit Trail):** "알바가 눈으로 봤다"가 아닌 "시스템 로그와 캡처본이 증명한다"는 논리로 인사 담당자의 심리적/법적 부담을 완벽히 제거.
2. **공수 제로 UX (Fallback Automation):** AI 오인식이나 서류 미비 시 발생하는 민원 업무를 '지원자 Self-Loop'로 해결하여 실제 운영 비용을 50% 이상 절감.
3. **독보적 신뢰 자산 (Parallel Capture):** 단순 결과값이 아닌 '증거 이미지'를 직접 생성하는 방식은 시장 내 유일한 기술적 해자(Moat)로 작용.

### ⚠️ 잠재 리스크 및 대응 (Risks)

1. **기관 사이트의 RPA 차단:** 주요 기관의 사이트 구조 변경이나 봇 차단 정책.
    - **대응:** API 우선 연동 방식을 고수하되, 사이트 변경 감지 시 72시간 내 스크립트를 자동 업데이트하는 모니터링 시스템 구축.
2. **AI 오판 시 배상 책임:** AI가 위조 서류를 통과시켰을 때의 책임 소재.
    - **대응:** 시스템은 '경고'와 '증거'만 제공하고 최종 확정 버튼은 '인간 실무자'가 누르게 설계하여 법적 배상 리스크를 분산(Human-in-the-loop).

### 🚀 단계별 실행 로드맵 (Roadmap)

- **Phase 1 (MVP):** 상위 3대 자격/학위 서류 대상 'OCR + 병렬 캡처' 엔진 출시. 채용 대행사 3곳 대상 PoC 진행 (리드타임 80% 단축 증명).
- **Phase 2 (Growth):** 카카오알림톡 Self-Loop 및 감사 리포트 자동 생성 기능 탑재. 주요 ATS 플랫폼(그리팅 등)과 API 연동하여 민간 기업 대량 침투.
- **Phase 3 (Standardization):** ISMS 인증 완료 후 931개 공공기관 전수조사 시장 진입. '공정채용 표준 솔루션'으로 포지셔닝 및 해외 학위 검증 모듈 확장.
